
Smart Campus Tracker

📌 About the Project
Smart Campus Tracker is a web-based application designed to simplify campus management by providing a centralized platform for students and administrators. The system helps manage academic and campus-related activities efficiently.

💻 Technologies Used
HTML
CSS
JavaScript
Python
Django
SQLite

✨ Features
User registration and login
Student dashboard
Campus information management
Secure data storage
Interactive user interface
Easy navigation between modules
Real-time data updates

🎯 Objective
The main goal of this project is to improve campus management by creating a simple, efficient, and user-friendly system for handling student and administrative tasks.
